C‑Flow: Руководство для чайников

Введение

Добро пожаловать в систему C‑Flow! Этот проект помогает организовывать работу ИИ‑агентов и автоматизировать разработку. Ниже вы узнаете, как начать.

Добро пожаловать! Здесь мы научимся запускать C‑Flow и создавать свой первый рой.

Установка

  1. Убедитесь, что установлен Node.js версии 18 или выше.
  2. Выполните команду npx claude-flow@alpha init --force для инициализации.

Сначала установите Node.js и запустите команду инициализации. Это создаст нужные папки.

Быстрый старт

  1. Запустите npx claude-flow@alpha swarm "создай мне REST API".
  2. Дождитесь вывода результатов. Вы увидите отчёт о работе агентов.

Используйте команду swarm, чтобы быстро получить результат. Агент всё сделает сам.

Графический интерфейс

В каталоге c-flow-v2/gui есть простое приложение на Python. Запустите его командой python3 main.py. Оно позволяет вводить команды и видеть вывод прямо в окне.

GUI облегчает работу: просто введите команду и нажмите Run.

Получение справки

Для списка всех команд выполните claude-flow help или смотрите файл Commands Reference в документации.

Всегда можно вызвать help, чтобы посмотреть доступные команды.
